WhatsApp connection via Pairing Code

How to use:

  1. Enter your WhatsApp number (without +)
  2. Choose a bot name
  3. Click "Generate Code"
  4. Open WhatsApp → Settings → Linked Devices
  5. Enter the generated code
  6. Wait for connection confirmation
